Smart Solution

The Smart Balcontracker is a stainless steel skeleton construction, an embedded microcontroller and 2 linear actuators. The system can be installed on all types of balcony fences. The Smart Balcontracker enables the installation of solar modules of different sizes and performance. The installation instructions are very easy to learn, so anyone can install our smart tracker on the balcony.

Uniqueness

Thanks to the use of a unique algorithm for controlling the position of the sun relative to the solar panel and periodically adjusting the angle of the solar panel, the average annual productivity of the system increases by 60% compared to vertically installed panels. You can use the calculator National Renewable Energy Laboratory (NREL) use. The calculator allows you to compare expected productivity over the course of a year for different types of PV systems.

Monitoring

After the installation and initial setup are completed, the smart balcony tracker controller begins converting the measured parameters, and information about them is sent via Wi-Fi to a cloud server for data storage and processing.

Connecting and Configuring the Phone

After the smart balcony tracker has been connected to the Wi-Fi, it carries out all further work automatically.

Mobile App "IRES Monitor"

The mobile app “IRES-Monitor” offers the ability to monitor electricity generation and visualize historical data. The mobile app also contains installation instructions for the Smart balcony tracker. In the "Settings" area, the user can limit the maximum permissible elevation angle of the solar panel as well as the location using a geographical map balcony power plant.

Data Control

The system is initially configured using the “IRES-Monitor” mobile app. All information about electricity generation is stored free of charge on a cloud server and is available to the user at any time via a mobile application.
